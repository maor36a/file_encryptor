/**
@ This program will help you protect any sensetive data you have.
@ Gather all the sensetive data to .txt files, and follow the orders to encrypt/decrypt this information.
*/

Put all text files you want to encrypt/decrypt and this program to the same folder.
After running the program, choose between encryption and decryption options - A password will be requested either way.
If encryption is chosen, all files will be encrypted, and "_encrypted" will be appended to all text files names.
If decryption is chosen, all files with the suffix "_encrypted" will try to be decrypted, and the suffix will be removed is succeded.
